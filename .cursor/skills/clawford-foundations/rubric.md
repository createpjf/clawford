# Clawford Foundations Rubric

Score each category on a 0-2 scale.

- `0`: failed or missing
- `1`: partially correct but incomplete
- `2`: correct, operational, and evidence-aware

Maximum score: `10`

## Category 1: Memory Hygiene

Checks:

- distinguishes memory from temporary context
- avoids persisting noise or one-off details
- identifies stable facts worth keeping

## Category 2: Lessons Discipline

Checks:

- treats lessons as reusable process rules
- requires evidence for durable lessons
- avoids unsupported self-reflection

## Category 3: Verification

Checks:

- includes concrete validation steps
- does not claim success before proof
- includes a replan path when checks fail

## Category 4: Multi-Agent Design

Checks:

- uses at least three meaningful roles when appropriate
- gives each role a clear responsibility
- describes compact, useful handoffs

## Category 5: Clarity And Operational Quality

Checks:

- answers are specific, not vague
- reasoning is actionable
- constraints and assumptions are named clearly

## Interpretation

- `9-10`: pass with strong Clawford fluency
- `7-8`: pass, but revisit weak categories
- `5-6`: borderline; reteach the failed module and retest
- `0-4`: fail; learner needs structured re-onboarding
