# Lesson 2 — 七种经典胶片模拟 / The 7 Classic Film Simulations

## 课程目标 / Learning Objective

记住并区分富士全部 7 种经典胶片模拟，能根据拍摄场景选择最合适的风格。

## 完整图谱 / Complete Map

### 🥇 ACROS — 高质感黑白
- **特点：** 富士最经典的黑白模式。深邃的黑色、丰富的灰阶、细腻的颗粒感
- **适用：** 建筑、街头、纹理感强的高对比场景、人像
- **不适合：** 需要色彩信息的场景
- **常见变体：** ACROS+R（红滤镜）、ACROS+G（绿滤镜）、ACROS+Ye（黄滤镜）
- **菜单描述：** 高质感黑白，适合纹理与图形感场景

### 🥈 Classic Chrome — 低饱和纪实
- **特点：** 低饱和、柔和对比、略带灰色调的纪实风格。颜色偏暖、偏土。
- **适用：** 街头摄影、编辑风、城市纪实、旅行记录
- **不适合：** 需要色彩鲜艳的场景（会被压得很平）
- **菜单描述：** 低饱和纪实感，适合街头与编辑风

### 🥉 Classic Neg. — 怀旧负片
- **特点：** 模仿彩色负片的感觉。阴影更密、色彩更暖、高光略带偏色
- **适用：** 日常生活感、怀旧氛围、人物与生活记录
- **不适合：** 商业/产品需要中性色彩的场景
- **菜单描述：** 怀旧负片气质，暗部更密，生活感更强

### 4️⃣ ETERNA — 电影感
- **特点：** 富士电影摄像机使用的预设。低反差、低饱和、高光柔和
- **适用：** 视频截图、安静氛围、夜景、柔和日光
- **不适合：** 需要 punch 和冲击力的场景（会显得太平）
- **菜单描述：** 低反差电影感，适合安静氛围和夜景

### 5️⃣ ASTIA / Soft — 柔和人像
- **特点：** 柔和、讨喜、人像友好。高光和阴影都偏软
- **适用：** 人像、晴天户外、柔和光线、产品
- **不适合：** 需要高对比或纪实质感的场景
- **菜单描述：** 柔和耐看，适合人像与温和日光

### 6️⃣ PROVIA / Standard — 自然中性
- **特点：** 均衡、自然、接近人眼所见。是富士的"默认"预设
- **适用：** 日常记录、参考底片、需要中性色彩的场景
- **不适合：** 想表达强烈风格的时候（太中性=没风格）
- **菜单描述：** 自然中性，适合日常和参考底片

### 7️⃣ Velvia / Vivid —  Punch 高饱和
- **特点：** 高饱和、高反差、高锐度。风景摄影的经典之选
- **适用：** 风光、旅行、自然摄影、需要强烈视觉冲击的场景
- **不适合：** 人像（肤色容易偏红/偏洋红）、纪实、安静氛围
- **菜单描述：** 高饱和高存在感，适合风光和旅行场景

## 选择指南 / Selection Guide

| 场景 | 推荐风格 | 原因 |
|------|---------|------|
| 高对比黑白 | ACROS | 纹理与图形感 |
| 街头纪实 | Classic Chrome | 低饱和压平色彩 |
| 日常生活感 | Classic Neg. | 暖调怀旧 |
| 电影感/安静 | ETERNA | 低反差柔和 |
| 人像（柔光）| ASTIA | 柔和讨喜 |
| 自然中性参考 | PROVIA | 均衡自然 |
| 风光/旅行 punch | Velvia | 高饱和冲击力 |

## 亮度与对比度的影响 / Brightness & Contrast Impact

 FujiDay 会分析输入图片的 **亮度** 和 **对比度**，据此推荐：

- **高亮度（>110）+ 高对比度（>60）：** 推荐 ACROS 或 Classic Chrome（需要控制高光）
- **低亮度 + 低对比度：** 推荐 ETERNA 或 Classic Neg.（提升反差和氛围）
- **高饱和主色：** 推荐 Velvia（进一步强化）
- **低饱和场景：** 推荐 Classic Chrome 或 ACROS（不依赖色彩）
