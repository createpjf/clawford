# Lesson 1 — 什么是 FujiDay？ / What is FujiDay?

## 课程目标 / Learning Objective

理解 FujiDay 的核心使命：让任何人都能给数码照片赋予富士胶片的独特美学。

## 什么是胶片模拟？/ What is a Film Simulation？

数码相机的"胶片模拟"（Film Simulation）是厂商用数字算法模拟经典胶片冲洗效果的预设。

富士 Fujifilm 以胶片模拟闻名，因为富士曾是胶片巨头，拥有 80+ 年的色彩科学积累。

**富士代表性的胶片模拟：**
- **ASTIA** — 柔和、人像友好
- **Classic Chrome** — 低饱和、纪实感、编辑风
- **PROVIA** — 自然中性、参考级
- **Velvia** — 高饱和、风光学
- **ETERNA** — 电影感、低反差
- **Classic Neg.** — 怀旧负片感
- **ACROS** — 高质量黑白

## FujiDay 是什么？/ What is FujiDay?

FujiDay 是富士胶片调色系统，让 AI agent 能够：

1. **分析图片** — 判断亮度、对比度、色彩分布、场景类型
2. **推荐风格** — 根据场景推荐最合适的胶片模拟
3. **生成配方** — 产出结构化的调色参数（film sim + WB + DR + Highlight + Shadow + Grain）
4. **渲染预览** — 输出调色后的图像预览
5. **Alex Webb 构图** — 进阶构图裁切，链入调色工作流

## FujiDay 的局限 / FujiDay's Limitations

FujiDay 明确声明：**这是近似调色，不是相机直出**。

它不能 100% 复现富士相机的机内 JPEG 处理引擎，但它提供：
- 接近的色调和色彩偏移
- 合理的动态范围控制
- 胶片颗粒感模拟
- 实用的照片风格输出

## 核心概念速查 / Quick Reference

| 概念 | 解释 |
|------|------|
| Film Simulation | 模拟特定胶片的色彩和影调特性 |
| Recipe | 结构化调色参数集 |
| DR (Dynamic Range) | 动态范围，DR200/DR400 控制高光细节保留 |
| Highlight/Shadow | 高光和阴影偏移，-2 到 +2 |
| WB Shift | 白平衡偏移，R/B 各 ±9 |
| Color Chrome | 色彩强化，对高饱和色彩增加层次 |
