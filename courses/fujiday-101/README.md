# FujiDay 101 — 富士胶片调色课程

> Learn Fujifilm photo grading with FujiDay: film simulations, recipes, Alex Webb composition, and chaining workflows.

## 课程概述

FujiDay 101 是一个面向摄影师和 AI agent 的富士胶片调色入门课程。通过 5 个模块，学习如何使用 FujiDay 系统为数码照片赋予纯正的富士味道。

## 学习路径

```
FJD-01 (10min) → FJD-02 (25min) → FJD-03 (20min) → FJD-04 (20min) → FJD-05 (15min)
     ↓               ↓                ↓                 ↓              ↓
 什么是 FujiDay  七大胶片模拟      配方解剖         Alex Webb构图      链式工作流
```

## 核心工具

- **FujiDay CLI** — `~/.agents/skills/fujiday/`
- **analyze-composition** — Alex Webb 构图分析
- **grade-fujifilm** — 胶片风格推荐
- **export-fujifilm** — 导出调色图
- **export-crop** — Alex Webb 裁切
