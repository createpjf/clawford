## Question 1 — 胶片模拟匹配（10分）

场景描述：海滩日落，天空橙红、海面金色，人物剪影。

请从 7 种胶片模拟中选择最合适的 2 个选项，并解释为什么。

**评分标准：**
- 10分：正确答案 + 清晰解释
- 7分：答案正确但解释模糊
- 5分：部分正确
- 0分：错误答案

## Question 2 — 配方解读（20分）

以下 FujiDay recipe 用于一张高光过曝的城市建筑照片：

```json
{
  "base_film_simulation": "Classic Chrome",
  "dynamic_range": "DR400",
  "highlight": "-2",
  "shadow": "0",
  "color": "-1",
  "grain": "Weak",
  "wb": "Auto"
}
```

请解释：
1. 为什么选择 DR400？（5分）
2. highlight 设置为 -2 起到什么作用？（5分）
3. 这张照片的整体色调预期是什么？（5分）
4. 如果想让阴影更重，应该把 shadow 调到什么值？（5分）

## Question 3 — Webb Fit 判断（20分）

描述一张适合 webb_risky 裁切的图片特征，包括：边缘压力、层叠感、色彩张力、光影分割感。

然后判断以下描述的 Webb Fit Score 区间（高/中/低）：

> 傍晚街头，一个红衣女子站在蓝色墙壁前，光线从右侧斜射，左侧有大片深色阴影，背景有霓虹灯。

**评分标准：**
- 20分：特征描述完整 + 判断正确 + 理由充分
- 15分：特征描述较完整 + 判断正确
- 10分：判断正确但理由不足
- 5分：判断错误
